🎮 Guía de Instalación y Optimización de Wine para Warcraft III en Archcraft

Guía paso a paso desde cero absoluto para configurar una botella de Wine de 32 bits optimizada para el launcher de Atinad, solucionando de raíz los problemas de precisión y retraso del mouse.

1️⃣ Activar los 32 bits (Multilib)

Abre la configuración de pacman para permitir repositorios de 32 bits necesarios para Wine:

sudo nano /etc/pacman.conf

💡 Nota: Baja con las flechas del teclado, bórrales el símbolo # a la línea [multilib] y a la línea de abajo Include = /etc/pacman.d/mirrorlist. Guarda presionando Ctrl + O (Enter) y sal con Ctrl + X.

2️⃣ Actualizar el sistema completo

Sincroniza los nuevos repositorios y aplica los cambios:

sudo pacman -Syu

3️⃣ Instalar Wine, Winetricks y componentes

Instala el motor de Wine junto a las herramientas de desempaquetado de Microsoft:

sudo pacman -S wine winetricks cabextract wine-mono wine-gecko

4️⃣ Crear la base de Wine en 32 bits

Fuerza al sistema a generar la botella por defecto en entorno de 32 bits para máxima compatibilidad con juegos clásicos:

export WINEARCH=win32 && winecfg

💡 Nota: En cuanto veas aparecer la ventana visual de configuración de Wine en tu pantalla, simplemente ciérrala. Esto concluirá el proceso de creación en la terminal de forma correcta.

5️⃣ Instalar Fuentes, DirectX 9 y .NET 4.0 (Modo Silencioso)

Instala las librerías necesarias sin asistentes ni pantallas emergentes molestas:

winetricks -q corefonts && winetricks -q d3dx9 && winetricks -q dotnet40

⏳ Nota: El componente de .NET Framework 4.0 es pesado y tardará varios minutos. Deja que la terminal trabaje tranquila y no la cierres hasta que vuelva a aparecer tu usuario de comandos.

6️⃣ Configurar el Mouse en Modo Nativo

Abre el panel de control de Wine para inyectar la librería directa de hardware:

winecfg

Pasos a seguir en la interfaz:

  1. Entra en la pestaña llamada Librerías (o Libraries).
  2. En la casilla "Nueva sustitución para la librería", escribe: dinput8
  3. Haz clic en el botón Añadir.
  4. Busca dinput8 en la lista inferior, selecciónala y dale a Editar.
  5. Marca la casilla que dice Nativa (Windows) y pulsa Aceptar.
  6. Dale al botón Aplicar y finalmente a Aceptar para cerrar el panel.

🛠️ Comandos Útiles de Mantenimiento

Abrir el Registro de Windows (Para calibrar la sensibilidad o MouseSensitivity):

wine regedit

Abrir el panel visual gráfico de Winetricks:

winetricks --gui

🚀 Comando Definitivo para Lanzar el Juego

Para jugar con el cursor 1 a 1 y completamente anclado a la resolución interna, sitúate en la carpeta de tu launcher mediante la terminal y ejecuta:

WINE_MOUSE_WARP_OVERRIDE=force wine ./Atinad.exe

⚙️ Optimización recomendada: Asegúrate de marcar el Modo DirectX 9 en el propio menú del launcher de Atinad. Al combinarlo con esta configuración, lograrás el rendimiento gráfico más fluido y estable en Linux.
